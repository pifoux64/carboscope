== URGENT : Please Upgrade to MaxiCharts ==

Gravity Forms Charts Reports has become MaxiCharts + MaxiCharts Gravity Forms Source Add-on ! :)
To follow using its features freely and transparently, please:
* Install [Maxicharts](https://wordpress.org/plugins/maxicharts/ "MaxiCharts")
* Install [Maxicharts Gravity Forms source add-on](https://wordpress.org/plugins/maxicharts-gravity-forms-source-add-on/ "GF Add-on")
* Deactivate and delete completely Gravity Forms Charts Reports

Additionnal add-ons have been added in case you need it:
* [MaxiCharts CSV add-on](https://wordpress.org/plugins/maxicharts-csv-source-add-on/ "CSV add-on") : in order to graph CSV files
* [MaxiCharts Gravity View add-on](https://wordpress.org/plugins/maxicharts-gravity-view-add-on/ "Gravity View add-on") : in order to field entries on "approved" ones
